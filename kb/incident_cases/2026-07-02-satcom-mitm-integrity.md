# 인시던트 2026-07-02 — SATCOM 링크 MITM 무결성 훼손 (정탐)

S3 룰(uav_satcom_integrity_fail) 알람. UAV-SATCOM 링크에서 MAC 검증 실패율이
정상 0.1% 대비 12%로 급등, 동시에 페이로드 체크섬 불일치 다발. 링크 경로 중간자
(AiTM, ICS T0830) 개입으로 원격측정 변조 시도 확인 → 정탐.

대응(PB-LINK-INTEG-03): 무결성 실패 세그먼트 격리 → 세션 키 롤오버 → 대체 링크
전환. 교훈: MAC 검증 실패율·체크섬 불일치는 단독으로는 기상/장비 노이즈와 겹치나,
**둘 동시 발생 + 지속**이면 링크 무결성 훼손 강신호. 예정 점검창(SATCOM-MAINT)
재접속과 구분할 것 — 점검창 이벤트는 무결성 실패를 동반하지 않는다.

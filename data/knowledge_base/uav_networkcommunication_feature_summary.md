# UAV NetworkCommunication: GPS Dataset Feature Summary (from raw data)

Source: raw files under `raw_data/uav_networkcommunication/` (UAV NetworkCommunication
GitHub dataset). This summary was generated by parsing the actual pkl/pcap contents
with scapy on 2026-06-11.

## gps_dataset.pkl (baseline / normal GPS traffic)

- Format: Python list of `(scapy IP packet, label)` tuples
- Samples: 800
- Label distribution:
- label `0`: 595 samples (74.4%)
- label `1`: 205 samples (25.6%)
- Each packet: `127.0.0.1 -> 127.0.0.1` UDP, payload 16000 bytes
  = 2000 float64 values (GPS baseband/feature vector per sample)
- Example packet: `IP / UDP 127.0.0.1:domain > 127.0.0.1:domain / Raw`

## SOC relevance

- The pkl labels make this dataset directly usable as ground truth for a
  GPS-spoofing/jamming detector feeding the UAV SOC pipeline.
- Canonical event mapping: a positive label corresponds to the
  `incident_case_gps_spoofing_gcs_normal` pattern (GPS anomaly while the GCS
  network link stays healthy), MITRE ATT&CK for ICS Wireless Compromise (T0860).

## 📁 제출 서류

### 1️⃣ 예선 보고서 (필수)

- 형식: PDF, 최대 50MB
- 파일명: `DAH2026_예선보고서_[팀명].pdf`
- 분량: 본문 기준 25~40페이지 (A4) 권장
- 언어: 한국어 또는 영어 (혼용 시 한국어 기준)
- 방산 분야 UAV/UGV 공격 시나리오에 대한 방어 AI 보고서

### 2️⃣ 보조 자료 (선택)

- 소스 코드 + 실행 환경 설명서
- Google Drive / Dropbox 등 외부 링크로 제출
- 공유 설정: **"링크가 있는 모든 사용자"** 로 설정
- 파일명: `DAH2026_소스코드_[팀명].zip`
**권장 폴더 구성**

```
README.md          ← 실행 방법, 의존성, 환경 변수
src/               ← 에이전트 코드
requirements.txt 또는 Dockerfile
docs/              ← 아키텍처 다이어그램, 데모 영상 링크
```

### 3️⃣ 부문 증빙 서류 (필수)

- 파일명: `DAH2026_부문서류_[팀명].pdf`
- 팀원 전원 서류를 **통합 PDF 1부**로 제출

| 부문 | 제출 서류 |
|---|---|
| 학생 | 재학증명서 / 졸업예정증명서 / 휴학증명서 / 교수추천서 중 택1 |
| 일반인 | 재직증명서 (프리랜서·연구원 등은 신분증 사본 또는 활동 증빙 대체 가능) |

## 🏆 예선 채점 기준 (100점 만점)
